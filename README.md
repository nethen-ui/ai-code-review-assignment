Installation

To run the code locally, follow the steps below:

1. Clone the repository or download the ZIP file
   
  ```bash
git clone https://github.com/nethen-ui/AI-Generated-Code-Review
  ```

2. Navigate into the repository directory:

  ```bash
cd AI-Generated-Code-Review
```

3. Run the code in each task using Python:

   - For Windows
     
   ```powershell
   python correct_task1.py
   python correct_task2.py
   python correct_task3.py

   ```
   - For Linux
     
   ```bash
   python3 correct_task1.py
   python3 correct_task2.py
   python3 correct_task3.py
